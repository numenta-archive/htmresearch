# Dimensionality reduction

Demo scripts to project SDRs in 2D, in order to make it 
possible to visualize clusters of SDRs. 

## Pre-requisites

Install pyclustering: 
```
    git clone https://github.com/annoviko/pyclustering
    cd pyclustering
    pip install . --user
```
